# 📊 Project Details

## Objective

Build an enterprise-style ETL pipeline using Alteryx.

---

## Workflow Overview

### 1. Data Ingestion
Loaded multiple data sources.

---

### 2. Data Integration
Sequential joins performed to combine datasets.

---

### 3. Data Cleaning
Removed duplicates and selected relevant columns.

---

### 4. Transformations
Created business metrics:

- Revenue  
- Discount Value  
- Net Sales  

---

### 5. Aggregation
Generated insights:

- Category Sales  
- Country Sales  
- Top 10 Customers  

---

### 6. Output
Exported final datasets into CSV format.

---

## Key Learnings

- Data integration across multiple tables  
- ETL pipeline design  
- Business transformation logic  
- Analytical summarization
